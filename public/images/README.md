# Photo Instructions

Please add your couple photos to this folder!

## File Naming
- `couple1.jpg`
- `couple2.jpg`
- `couple3.jpg`
- Add more if you want!

## Recommended Settings
- **Format**: JPG, PNG, or WebP
- **Size**: Around 1200x800px
- **File size**: Under 1MB each for fast loading
- **Orientation**: Landscape works best for the carousel

## Tips
- Choose your best photos together
- Mix different moments (dates, trips, candid shots)
- Make sure photos are well-lit and clear

Once you add your photos, update the paths in:
`src/components/MemoriesGift.jsx`
